Greater Trochanter Implant

Greater Trochanter Implant

This is a model of a Greater Trochanter Implant, designed for use in hip arthroplasty. The implant is designed to be used in cases of severe bone loss in the greater trochanter. For more details, see the project README.

Project Overview

  • Medical device design for greater trochanter fracture fixation
  • Anatomically contoured for optimal fit
  • Biocompatible titanium alloy construction
  • Multiple fixation points for secure attachment

Rover Arm Mechanism

Mars Rover Arm Mechanism (1:1 Replica)

This is a 1:1 non-functional replica of the arm mechanism from the Mars Opportunity Rover. It was modeled to study the mechanical design and kinematics. For more details, see the project README.

Project Overview

  • Accurate 1:1 scale reproduction of Opportunity rover arm
  • Educational display model for museum exhibition
  • Faithful recreation of all visible components
  • Historical accuracy and educational value
